# sk.CheckFormData

## 干啥的
用于检查表单数据是否都已填写完毕，没则弹窗提示并且取消表单提交

## 使用方法
1.引用js文件 sk.CheckFormData.js
1.在你需要检查的form元素添加class：sk-form-check-auto。  
>  更多使用方式  [请点我点我](#baidu.com)  。

## 特点
- 简单易用
- 适合小网站使用
- 不依赖jquery库
- 浏览器兼容性好
